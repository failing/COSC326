Completes the specification set out in Etude 5.

To compile:
	javac *java
To run: 
	java Main

It will display a line with 
"scale r g b"

Enter your values line by line for example:
	1.0 255 0 0
	0.8 0 255 0
	0.1 0 0 255

and then press control-d.

A window in the top left corner will display the corresponding squares in a window.
